Projection Studio for Web

By Eric Burke

Projection Studio for Web is a free application designed for photographers to use during light projection photoshoots.

If you publish photos using this app, I appreciate being credited, although this is not required.

Start using Projection Studio for Web

This app only runs on desktop web browsers and does not work on mobile or tablets.
